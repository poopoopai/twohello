(1) git branch "增加修改黨案"

(5)git commit -am "大該說明更改的內容"

(7)然後在pr momo到master

(8)刪掉github上的分支

git checkout --.空白 返回原本的檔

git fetch "合夥人的url" 檢查版本更新
